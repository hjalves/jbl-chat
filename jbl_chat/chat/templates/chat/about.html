About

This is a simple real-time chat application built with Django and HTMX. While it provides basic messaging functionality, it is not intended to be a full-featured chat platform.

Messages cannot be edited or deleted, and private chats are not supported. Additionally, new user registration is not yet available. To create an account, you must use the Django admin interface or the Django shell.

This project was developed by Humberto Alves as part of a hiring process to demonstrate how to build a real-time chat application with Django and HTMX.
